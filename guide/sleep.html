# sleep

使async内部程序等待一定时间后再执行

参数 说明 类型 必填值 默认值
time 等待时间 Number true 0

# Demo:

基本用法

import { sleep } from 'lonjin-helper'

async function fn() {
    await sleep(3000)
    console.log('hello world')
}
